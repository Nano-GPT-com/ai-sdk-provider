# NanoGPT

[NanoGPT](https://nano-gpt.com) delivers lightweight, efficient language models designed for production workloads. This community provider brings NanoGPT support to the [Vercel AI SDK](https://ai-sdk.dev) so you can reuse the same abstractions across providers.

> **Note:** The implementation is currently under active development. Follow the repository for updates and final API surface details.

## Provider Instance

```typescript
import { createNanoGPT } from '@nanogpt/ai-sdk-provider';

const nanogpt = createNanoGPT({
  apiKey: process.env.NANOGPT_API_KEY!,
});
```

You can generate an API key from the [NanoGPT dashboard](https://nano-gpt.com/dashboard) (URL TBC).

## Model Usage

```typescript
import { generateText } from 'ai';
import { createNanoGPT } from '@nanogpt/ai-sdk-provider';

const nanogpt = createNanoGPT({ apiKey: process.env.NANOGPT_API_KEY! });

const { text } = await generateText({
  model: nanogpt.chat('nanogpt/chat-v1'),
  prompt: 'Summarize the NanoGPT mission statement.',
});
```

Streaming support, embeddings, and tool-calling examples will be documented here once the endpoints are finalized.
